Pattern - RDF List

This wizard creates an RDF list whose elements are restricted to individuals of a given class.

RDF lists can be useful to retain order, but have no semantics in OWL and therefore cannot be reasoned with.

eg. An RDF list could be used to describe processes that must follow a particular order. Remember, the list cannot be reasoned with using DL reasoners.

The wizard will create:

  • A subclass of RDF list that can only contain individuals from a certain class
  • A property that can be used to create restrictions with this list
  • Optionally, a restriction on a given class using this list
